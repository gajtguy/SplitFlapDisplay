## Project Overview
A modular split-flap display system with 64 characters per module, controllable via RS485 from a Raspberry Pi. Designed by Adam G Makes.

## Assembly Notes

1. The firmware uses RS485 half-duplex communication at 115200 baud
2. Each module has a unique ID (0-44) stored in EEPROM
3. The module homes using the Hall effect sensor on boot
4. Character positions are stored in EEPROM for accuracy
5. The stepper uses half-step sequences for smooth operation
